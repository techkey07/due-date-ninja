# due-date-ninja
Due Date Ninja is a Python console app for managing assignments by tracking due dates and automating priority levels. It lets you add, view, and complete tasks while showing motivational messages. The app also provides stats and charts to track your progress. Simple and effective for beginners learning Python and task management

Due Date Ninja is a Python console application designed to help students and professionals effectively manage their assignments by tracking due dates and priorities.

Features:

Add Assignments: Quickly add new assignments with a title, subject, and due date. The app validates input to prevent errors.

Automatic Priority Calculation: Assignments are automatically categorized as high, medium, or low priority based on how soon their due date is, allowing you to focus on urgent tasks.

View Schedule: View a sorted list of all pending assignments, displaying how many days remain and their priority. Each task is accompanied by motivational and funny messages to keep you engaged.

Mark Completed: Easily mark assignments as completed to keep your task list up to date and reduce clutter.

Statistics and Visualization: Get a summary of total, completed, and pending assignments along with visual bar charts showing the priority distribution and overall progress using Matplotlib.

User-Friendly Console Interface: Navigate the app easily with a simple menu-driven console interface.

Use of Python Libraries: Utilizes Python's datetime module for date handling, random for varied messages, and matplotlib for generating charts.

This project is perfect for beginners looking to practice Python programming, date manipulation, and simple data visualization while managing tasks effectively
